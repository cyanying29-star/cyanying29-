產品定位

結合「退休缺口試算」、「理財教育」、「兼職媒合」與「B2G 協作」的整合型服務，目的在於：

  • 讓使用者快速看懂退休缺口並獲得可行改善策略
  • 提供可落地的補強方案（短期補貼、中期職涯、長期投資）
  • 建立社群支持，減少心理/社會成本
試算器
個人化退休缺口估算與行動建議

根據風險偏好給出配置建議

兼職、講座、社區服務的媒合平台
自助申請
線上補貼/福利申請與表單範本
數據儀表板
匿名化需求熱區給政府/機構
社群與心理支持
互助論壇與專家諮詢

商業模式與合作

  • B2G：白標服務、資料報表與需求分析
  • B2B：對保險、銀行推薦進階理財與保險方案抽成
  • B2C：會員制進階功能（AI 教練、專家諮詢）
  • CSR / 贊助：企業贊助就業媒合專案

產品路線圖（建議）

0-2 個月
MVP：試算器 + 資訊整合頁面 + 基本註冊

技術建議與堆疊

  • 前端：React Native（跨平台）或 Flutter
  • 後端：Firebase / Supabase（認證、資料、通知）
  • AI：OpenAI / LLM 做智能問答與理財教練
  • 資料：串接政府 Open Data 與機關 API
